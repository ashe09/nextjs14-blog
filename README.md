# Next.js 14 & Sanity Blog
Next.js14とSanityで簡易的なブログを練習として作成しました。

## Getting Started
Next.js: Run the development server

```bash
npm run dev
```

Sanity: First, change directory to Sanity.

```bash
cd sanity
```

Next, run the development server

```bash
npm run dev
```

## Environment
- Next.js 14
- React
- TypeScript
- Sanity
- TailwindCSS
